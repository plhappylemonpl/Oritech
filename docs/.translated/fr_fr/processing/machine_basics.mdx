title: Informations Générales
icon: oritech:assembler_block
---

Toutes les machines Oritech nécessitent de l'énergie pour fonctionner.

Le côté supérieur est uniquement destiné à l'insertion d'objets, tandis que les côtés peuvent être utilisés pour insérer et extraire. Le bas permet uniquement l'extraction d'objets.

Les objets contenus dans une machine seront toujours lâchés lorsque la machine est détruite, tandis que l'énergie et les liquides qu'elle contient seront perdus. Les machines peuvent être cassées avec une pioche. Il n'est pas nécessaire d'utiliser une clé spéciale ou un outil similaire.
