---
title: Assembleur
icon: oritech:assembler_block
---

L'assembleur possède une grille de craft 2x2 et assemblera automatiquement les objets tant qu'il dispose d'énergie, d'objets dans les emplacements d'entrée, et d'espace pour placer le produit fini dans l'emplacement de sortie.

Certains composants de machine peuvent être fabriqués sans assembleur, mais les fabriquer avec l'assembleur consomme moins de ressources.
