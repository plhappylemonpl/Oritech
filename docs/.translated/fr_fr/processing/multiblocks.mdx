Certaines machines nécessitent des noyaux de machine supplémentaires pour fonctionner. Ceux-ci doivent être placés à des positions spécifiques par rapport à la machine elle-même. La quantité de noyaux nécessaire pour une machine est indiquée dans l'infobulle de la machine. Les noyaux ne peuvent pas être partagés entre plusieurs machines.

Pour commencer la construction de la structure multibloc, placez d'abord la machine elle-même.

En faisant un clic droit sur la machine, tous les blocs nécessitant des noyaux seront mis en surbrillance pendant quelques secondes. Vous pouvez ensuite y placer les noyaux. Alternativement, vous pouvez faire un clic droit sur la machine avec un noyau dans la main, ce qui placera automatiquement le noyau suivant. 

Notez que cela ne fonctionne que si les emplacements cibles ne sont pas obstrués. Si un obstacle est présent, le bloc sera mis en surbrillance.

Chaque noyau de machine possède une qualité associée. Utiliser de meilleurs noyaux de machine augmente la qualité totale de la machine.

Pour voir la qualité résultante d'une machine, survolez l'icône de la machine en haut à gauche de 

l'interface utilisateur. La qualité de la machine elle-même n'affecte pas directement le fonctionnement de la machine, mais elle détermine combien de modules peuvent être ajoutés à une machine. Si vous ne prévoyez pas d'ajouter des modules, vous pouvez utiliser les noyaux les moins chers.
