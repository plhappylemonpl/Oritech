---
title: Les âmes
icon: oritech:enchantment_catalyst_block
---

Lorsqu'une entité vivante meurt, elle libère une âme. Si un bloc capable de collecter l'âme est à portée (16 blocs), l'âme sera collectée par ce bloc.

Cela peut être vu comme une petite âme volant vers le bloc.

Les âmes peuvent être collectées par :
- Un Catalyseur arcanique
- Un Contrôleur de génération
