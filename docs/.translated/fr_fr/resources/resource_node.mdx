Les gisements de ressources se trouvent partout dans le monde, apparaissant en clusters juste au-dessus de la couche de bedrock. Il existe des gisements de ressources pour tous les types de minerais de l'Overworld. Ils sont généralement recouverts d'une petite poche de [pétrole brut](../resources/crude_oil).

Les gisements de ressources ne peuvent être minés qu'avec un [extracteur de bedrock](../interaction/deepdrill).

À la surface, la localisation d'un cluster de gisements de ressources peut être identifiée par la présence d'[Amas de minerais](../resources/ore_boulder). Le bloc de minerai indique également le type de minerais présents dans le cluster.
