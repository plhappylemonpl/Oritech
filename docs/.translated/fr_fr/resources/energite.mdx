title: Energite
icon: oritech:energite_ingot
---

L'Energite est un alliage de [nickel](../resources/nickel) et de [fluxite](../resources/fluxite) qui ne peut être fabriqué que dans une [fonderie](../processing/foundry).

Il est tellement plein d’énergie potentielle que vous pouvez ressentir un léger picotement en le touchant.

Il est utilisé pour les batteries, les modules de machine et les composants servant à créer certaines machines qui génèrent ou stockent de l'énergie.
