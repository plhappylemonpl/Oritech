L'acier est un alliage de base utilisé pour créer les fils, bobines, moteurs et revêtements nécessaires à la construction de toutes les autres machines.

L'acier peut être créé dans une grille de craft :

    ]}

    count={1}
/>

La [fonderie](../processing/foundry) peut produire davantage d'acier en utilisant moins de fer et de charbon.
