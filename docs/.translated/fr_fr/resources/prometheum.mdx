Le Prometheum est un matériau avancé qui ne peut être produit que dans une [forge atomique](../processing/atomic_forge).

Il est utilisé pour fabriquer des [outils en prometheum](../tools/prometheum_tools) et le noyau de machine ultime (niveau 7).
