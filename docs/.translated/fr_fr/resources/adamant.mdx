L'Adamant est un alliage de base utilisé pour créer l'[assembleur](../processing/assembler), le [module de chaudière à vapeur](../logistics/steam), les noyaux de machine, ainsi que d'autres composants et outils.

L'Adamant peut être créé dans une grille de craft :

<CraftingRecipe
    slots={[
        'oritech:nickel_ingot',
        'oritech:nickel_ingot',
        '',
        'diamond',
        'diamond',
        '',
        '',
        '',
        ''
    ]}
    result="oritech:adamant_ingot"
    count={1}
/>

La [fonderie](../processing/foundry) peut produire davantage d'adamant en utilisant moins de diamant et de nickel.
