L'armure exo est une armure spéciale qui peut vous aider dans vos aventures.

- **Le casque** vous donne une vision nocturne. 
- **le plastron** recharge vos objets pour vous permettre de rester plus longtemps dans les mines..
- **le pantalon** vous aide à courir plus vite.
- **les bottes** vous rendent immunisé aux dégâts de chute.
