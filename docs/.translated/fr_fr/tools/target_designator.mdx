---
title: Sélecteur de Cible
icon: oritech:target_designator
---

Le désignateur de cible est un outil utilisé pour... désigner des cibles. Faites un clic droit avec "Shift" sur un bloc (ou une station de drone) pour désigner une cible, puis utilisez-le sur la [station de drone](../logistics/drone_port) ou le [laser enderique](../interaction/enderic_laser) pour leur attribuer les cibles.
