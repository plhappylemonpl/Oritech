Lorsqu'il est alimenté en RF, le Coupe-arbres coupera un arbre entier, y compris les feuilles. Le Coupe-arbres ne coupera les bûches qu'à une distance maximale de 7 blocs, il pourrait donc ne pas atteindre le sommet des grands arbres.

Les bûches, feuilles et objets tombés des feuilles seront stockés dans l'inventaire interne du coupeur d'arbres.
