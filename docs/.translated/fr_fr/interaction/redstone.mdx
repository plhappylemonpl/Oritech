---
title: Redstone
icon: oritech:energy_pipe
---

Certaines machines Oritech peuvent interagir avec la redstone, le réservoir portable et le stockage d'énergie portable. Le contenu du réservoir portable peut être mesuré avec un comparateur, et la sortie des stockages d'énergie portables peut être désactivée avec un signal redstone. 
Pour les autres blocs, un "Module d'unité de contrôle redstone" est requis.

Le module d'unité de contrôle redstone peut être attaché comme tout autre addon et configuré via l'interface utilisateur. Une fois configuré, les données peuvent être lues en utilisant un comparateur. Le signal du comparateur sera émis par le module, et non par la machine elle-même. 
La machine peut également être désactivée avec un signal redstone envoyé au module.
