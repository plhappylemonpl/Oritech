Utilisé pour fertiliser les cultures situées sous le [cadre de machine](../interaction/machine_frames) connecté. Nécessite de l'eau et de l'énergie pour fonctionner. L'effet sur les blocs en dessous est similaire à

l'application de poudre d'os, bien que moins puissant.

Il hydrate également les terres agricoles en dessous.
