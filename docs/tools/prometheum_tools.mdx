<ModAsset location="oritech:promethium_pickaxe" /> The prometheum pickaxe is an advanced, unbreakable, and enchantable tool that functions as a pickaxe.

The prometheum pickaxe features two modes: **silk touch** and **area** (1x3 mining). To switch between modes, hold shift and right-click.

<ModAsset location="oritech:promethium_axe" /> The prometheum battleaxe is another advanced, unbreakable, and enchantable tool that serves as an axe.
