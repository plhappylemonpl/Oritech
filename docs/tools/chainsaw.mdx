<ModAsset location="oritech:chainsaw" /> The chainsaw is a rechargeable and unbreakable tool that functions as an axe.
It can also be enchanted. To charge the chainsaw, place it in an [energy storage](../logistics/energy) block or wear an [exo chestplate](../tools/exo_armor).
